# 🚀 LISTO PARA TEST: Nordic Mystic Overhaul (Alpha 0.1.0)

¡El mod está preparado para su primer despliegue en la Grieta del Invocador! Sigue estos pasos exactos para activar el entorno nórdico.

## 🛠 Pasos en CSLOLManager

1.  **Importar el Mod:**
    *   Abre **CSLOLManager**.
    *   Arrastra la carpeta `mods/NordicOverhaul` directamente a la ventana del programa.
    *   Asegúrate de que aparezca con el nombre **"Nordic Mystic Overhaul"** y el icono híbrido de mármol.

2.  **Activar el Mod:**
    *   Marca la casilla junto al nombre del mod para habilitarlo.
    *   Haz clic en el botón **"Run"** (o el icono de play) en la parte superior para que CSLOLManager prepare los archivos para el juego.

3.  **Entrar al Juego:**
    *   Abre el cliente de League of Legends.
    *   Crea una **Herramienta de Práctica** (Practice Tool).
    *   Selecciona cualquier campeón y entra a la partida.

## 🔍 Qué verificar en este test

*   **Base Azul (Orden):** El suelo debe mostrar el **Mármol Blanco** con grietas azul hielo.
*   **Base Roja (Caos):** El suelo debe mostrar el **Mármol Negro** con grietas verde esmeralda.
*   **Interfaz (HUD):** El marco de habilidades debe tener la textura de **Madera de Fresno y Oro**.
