## 🧠 What it does / Qué hace

**[EN]**  
[Create folders based on the activities within the specified class, and reorganize the files you have created, as well as any notes or attachments within the activity that you did not create yourself.]

**[ES]**  
[Creacion de carpetas segun las actividades dentro de la clase especificada, junto con la reorganizacion de archivos que has creado y si hay apuntes o archivos adjuntos dentro de la actividad que no has creado tu.]

- [Selection of the class to organize / Seleccion de la clase de Classroom]
- [Reorganization of folders / Reorganizacion de las carpetas]
